### 5.2 表格


```ruby

姓名|技能|排行

--|:--:|--:

刘备|哭|大哥

关羽|打|二哥

张飞|骂|三弟


第一行为表头

第二行分割表头和内容。

- 有一个就行，为了对齐，多加了几个


文字默认居左

|:--:|两边加冒号,表示文字居中

|--: 右边加冒号,表示文字居右

```
